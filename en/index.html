Get Started

Open-closed Principle

The O of SOLID stands for Open-closed principle that means entities should be open for extension but closed for modification. With React Pluggable, we can add plugins and extend the system without modifying the existing files and features.

Declarative as well as Imperative

Declarative is beautiful but Imperative is powerful for extensibility. React Pluggable uses a mixed approach so that you can add or remove a feature with a plugin.

Thinking Features over Components

React abstracts components very well but a feature may have more than just components. React Pluggable pushes you to a feature mindset over component mindset.
