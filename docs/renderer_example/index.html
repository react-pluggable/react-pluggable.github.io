Renderer Example

Sometimes we want to render React components from the plugin. An example of this would be, say, you have a Dashboard app and you want to build a "Share" feature which shows a button in the header and clicking on which opens up a popup to show the share options. React Pluggable ships with a Renderer Plugin to solve this problem. We can use the RendererPlugin to define placeholders in our React app which can be used by the plugins to render React components.

import { RendererPlugin } from "react-pluggable";

Let us create a Plugin which will add a component

SharePlugin.tsx

import React from "react";
import { IPlugin, PluginStore } from "react-pluggable";

class SharePlugin implements IPlugin {

  getPluginName(): string {
    return "Share plugin";
  }

  getDependencies(): string[] {
    return [];
  }

  public pluginStore: any;

  init(pluginStore: PluginStore): void {
    this.pluginStore = pluginStore;
  }

  activate(): void {
    this.pluginStore.executeFunction("RendererPlugin.add", "header:after", () => (
      <button>Share</button>
    ));
  }

  deactivate(): void {
        // We need to speak to Aditya / Himanshu and verify this part
    this.pluginStore.executeFunction("RendererPlugin.remove");
  }
}

export default SharePlugin;

Look at the excuteFunction called by the PluginStore object in RenderPlugin.tsx. It consists of 3 arguments :

  1. First argument always contains a string ‘RendererPlugin.add’ which means we are executing the method correspond to RendererPlugin.add present in the RendererPlugin.

  2. Second argument contains the name of the key against which we want to store the component provided.

  3. Finally, third argument contains the component to add.

Note that each key stores the array of components against it which means that we can store multiple components for a particular key.

App.tsx

import React from "react";
import "./App.css";
import {
  createPluginStore,
  PluginProvider,
  RendererPlugin,
} from "react-pluggable";
import SharePlugin from "./components/plugins/SharePlugin";
import Header from "./components/Header";

const pluginStore = createPluginStore();
pluginStore.install(new RendererPlugin());
pluginStore.install(new SharePlugin());

function App() {
  return (
    <PluginProvider pluginStore={pluginStore}>
      <Header />
    </PluginProvider>
  );
}

export default App;

Important: One thing to note if you are using RendererPlugin then always install it first before any other plugins, otherwise it will not work.

Header.tsx

import * as React from "react";
import { usePluginStore } from "react-pluggable";

const Header = (props: any) => {
  const pluginStore: any = usePluginStore();

  let Renderer = pluginStore.executeFunction(
    "RendererPlugin.getRendererComponent"
  );

  return (
    <>
      <h1>I am header</h1>
      <Renderer placement={"header:after"} />
    </>
  );
};

export default Header;
